## huachat_new-app

这是一个新的独立“工作区目录”（新应用场景），与当前培训分析代码分开放置在 `workspaces/` 下。

你可以在 IDE 里直接“打开文件夹”到本目录，把它当成一个独立工作区使用。

### 快速开始

1) 创建并激活虚拟环境（可选但推荐）

```bash
python -m venv .venv
```

2) 安装依赖

```bash
pip install -r requirements.txt
```

3) 运行

```bash
python main.py
```

### 配置

配置文件：`config/config.yaml`
